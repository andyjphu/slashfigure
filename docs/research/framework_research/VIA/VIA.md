# VGG Image Annotator (VIA)

- **URL:** https://www.robots.ox.ac.uk/~vgg/software/via/
- **License:** BSD-2-Clause

Standalone image annotator (<400KB single HTML file). Separation of `shape_attributes` (geometry) from `region_attributes` (semantics) is an elegant pattern: separate "where/what shape" from "what it means."
