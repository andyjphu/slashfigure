# LLMermaid

- **URL:** https://github.com/fladdict/llmermaid (~73 stars)
- **License:** MIT

Uses Mermaid diagrams as structured execution plans for LLMs. Flips the relationship: diagrams guide LLMs instead of LLMs generating diagrams. Shows that Mermaid is rich enough to serve as a programming language for LLM behavior -- reinforcing it as well-suited for structured communication with LLMs.
