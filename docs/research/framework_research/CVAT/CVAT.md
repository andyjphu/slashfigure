# CVAT

- **URL:** https://github.com/cvat-ai/cvat (~15,552 stars)
- **License:** MIT

Computer vision annotation tool. Exports in COCO JSON, Pascal VOC XML, YOLO txt. COCO format: `{annotations: [{bbox: [x,y,w,h], category_id, segmentation}]}`. Adopting COCO-compatible annotation structures would make our metadata interoperable with the vision tools ecosystem.
