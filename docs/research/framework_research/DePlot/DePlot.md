# DePlot

- **URL:** https://huggingface.co/google/deplot
- **Paper:** ACL 2023 (arXiv:2212.10505)
- **License:** Apache-2.0

Google model converting chart images into linearized tables for LLM reasoning. Key finding: converting visual content to structured tabular text enables LLM reasoning that **outperforms end-to-end vision models by 29.4%**. Validates providing structured text metadata rather than relying on vision-only understanding.
