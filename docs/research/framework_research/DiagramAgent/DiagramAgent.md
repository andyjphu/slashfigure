# DiagramAgent

- **Paper:** CVPR 2025 (arXiv:2411.11916)
- **License:** Research

Four-module framework: Plan Agent, Code Agent, Check Agent, Diagram-to-Code Agent. Uses LaTeX and DOT as code representations. The Diagram-to-Code Agent (Qwen2-VL -> structured code) demonstrates the reverse of our pipeline. Together with our draw-to-metadata direction, enables full round-tripping.

Establishes that code-based diagram representations (LaTeX, DOT) are superior to pixel-based for generation and editing.
