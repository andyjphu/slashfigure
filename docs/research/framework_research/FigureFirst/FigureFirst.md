# FigureFirst

- **Full Name:** FigureFirst -- A Layout-first Approach for Scientific Figures
- **URL:** academia.edu paper

## What It Does
Users design layouts in Inkscape (SVG), then FigureFirst generates Matplotlib axes matching the layout. Data plotted programmatically, saved back into SVG. Iterative layout refinement without regenerating data.

## Key for Our Project
Demonstrates the "visual layout + programmatic data" workflow that scientists want. Our app could be the all-in-one tool that combines layout and data in a single interface, eliminating the Inkscape + Matplotlib roundtrip.
