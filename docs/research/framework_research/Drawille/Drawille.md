# Drawille

- **URL:** https://github.com/asciimoo/drawille (~3,228 stars) | **License:** AGPL-3.0

Unicode braille characters (U+2800-U+28FF) for terminal pixel graphics. Each character = 2x4 pixel grid = 8 pixels per character. 8x the resolution of standard ASCII art in the same text footprint. Compact spatial representation for LLMs.
