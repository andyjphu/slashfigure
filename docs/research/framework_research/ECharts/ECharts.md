# Apache ECharts

- **Full Name:** Apache ECharts
- **URL:** https://github.com/apache/echarts
- **Stars:** ~66,065
- **License:** Apache-2.0

## What It Does
Framework-agnostic charting. Canvas and SVG renderers. 20+ chart types including scientific ones (scatter, heatmap, parallel coordinates, graph/network, tree, sunburst, sankey).

## Key for Our Project
- **Option-as-JSON paradigm** -- entire chart state is one serializable JSON object. `chart.setOption(option)` / `chart.getOption()`.
- **Dual Canvas/SVG renderer** -- Canvas for interactive editing, SVG for export.
- **`dataset` component** for separating data from visual mapping.
- Apache-2.0 license, massive community.

## Key Insight
The option-as-JSON model is the most powerful scene description format among charting libraries. Aligns perfectly with metadata-driven approach.
