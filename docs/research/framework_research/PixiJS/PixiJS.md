# PixiJS

- **URL:** https://github.com/pixijs/pixijs (~46,800 stars)
- **License:** MIT

Fastest open-source 2D WebGL/WebGPU renderer. GPU-accelerated sprite batching. 60fps with thousands of objects. Modular extension architecture (v8). No built-in serialization -- you build the entire app layer. Best for extreme performance requirements but requires building everything (data model, serialization, metadata, tools) from scratch.
