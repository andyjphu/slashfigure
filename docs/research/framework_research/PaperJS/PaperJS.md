# Paper.js

- **URL:** https://github.com/paperjs/paper.js (~15,000 stars)
- **License:** MIT

Vector graphics scripting framework on Canvas. Full scene graph (Project > Layer > Group > Item). Every Item has a `data` property for arbitrary metadata (persists via `exportJSON()`). Boolean operations on paths (unite, intersect, subtract). Excellent for precise vector graphics. Main limitation: no modern framework bindings (no React).
