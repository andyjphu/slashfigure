# Label Studio

- **URL:** https://github.com/HumanSignal/label-studio (~26,927 stars)
- **License:** Apache-2.0

Multi-modal data labeling platform. Exports JSON with region annotations: `{regions: [{type, x, y, width, height, labels}]}`. Supports COCO, Pascal VOC formats. The region annotation format -- attaching typed, labeled regions with coordinates to a canvas -- is essentially what our metadata system produces.
