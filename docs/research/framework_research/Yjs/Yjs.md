# Yjs

- **Full Name:** Yjs -- Shared Data Types for Collaborative Software
- **URL:** https://github.com/yjs/yjs
- **Stars:** ~20,000+
- **License:** MIT

## What It Does
CRDT framework with shared data types (Map, Array, Text) that sync automatically. Offline editing, version snapshots, undo/redo, shared cursors. Transport-agnostic.

## Key for Our Project
Most mature JS CRDT library. If we need collaboration, this is the fastest path to market. Rich ecosystem of providers (WebSocket, WebRTC, IndexedDB).

## Alternative: Loro
https://github.com/loro-dev/loro -- Built in Rust with WASM bindings. Newer, better memory/perf. Uses Fugue algorithm. Consider if building with Rust/WASM stack.
