# Scientific Inkscape

- **Full Name:** Scientific Inkscape Extensions
- **URL:** https://github.com/burghoff/Scientific-Inkscape
- **License:** Check repo

## What It Does
Inkscape extensions for scientific figures:
- **Scaler:** Resize plots without modifying text/ticks (critical for multi-panel figures)
- **Autoexporter:** Auto-refresh SVG exports
- **Flattener:** Eliminate imported figure structure for easier editing
- **Homogenizer:** Uniform fonts, sizes, stroke widths
- **Text Ghoster:** Semi-transparent background behind text for data overlay

## Key for Our Project
These define must-have features for any scientific figure tool. The **Scaler concept** (resize without affecting text) is particularly important and should be a core feature. The Homogenizer (uniform styling across selection) is also essential for publication-quality figures.
