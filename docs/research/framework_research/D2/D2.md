## What It Does
Modern declarative diagramming language (2022) with clean, minimal syntax. Multiple layout engines. Software architecture diagrams.

## Key for Our Project
**Fault-tolerant, warning-first design** is ideal for LLM generation -- minor syntax errors produce warnings instead of failures. Nested container support maps well to grouped/layered canvas elements.

```
x -> y: hello world
container {
  a -> b
}
```
