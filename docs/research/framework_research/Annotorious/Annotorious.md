# Annotorious

- **Full Name:** Annotorious -- JavaScript Image Annotation Library
- **URL:** https://github.com/annotorious/annotorious
- **Stars:** ~829
- **License:** BSD-3-Clause

## What It Does
Client-side image annotation with rectangles and polygons, outputting W3C Web Annotation format with `SvgSelector` for shapes.

## Key for Our Project
Most directly relevant annotation library. Already handles UX of selecting regions and producing W3C-compliant JSON. `SvgSelector` output means annotation shapes are SVG -- directly compatible with an SVG-based canvas. Plugin architecture allows custom annotation types (equations, tables).
