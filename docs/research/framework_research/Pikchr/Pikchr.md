# Pikchr

- **URL:** https://pikchr.org/ / https://github.com/drhsqlite/pikchr
- **License:** 0-clause BSD
- **Author:** D. Richard Hipp (creator of SQLite)

PIC-like markup for diagrams in docs: `box "Hello" ; arrow ; box "World"`. Designed for Markdown embedding, safe for untrusted input. Generates all SQLite syntax diagrams. Simple enough for LLMs to generate, safe enough to render without risk. Excellent candidate for LLM-writable diagram description.
