# ASCIIFlow

- **URL:** https://asciiflow.com/ (open source)
- **License:** MIT

Interactive web-based ASCII diagram editor. The drawing IS the text -- what you see is a grid of characters. Demonstrates that interactive diagram editing with a text-native format is viable and intuitive. The drawing-to-text pipeline is trivial because the drawing IS text. This is the ideal to aspire to for LLM-friendliness.
