# Plotly.js

- **Full Name:** Plotly.js
- **URL:** https://github.com/plotly/plotly.js
- **Stars:** ~18,200
- **License:** MIT

## What It Does
High-level charting: 40+ chart types, interactive, publication-quality. SVG (default) + WebGL for large datasets.

## Key for Our Project
- **Declarative JSON config** -- `{data, layout, config, frames}` is fully serializable. The Plotly JSON schema is a published standard.
- **`layout.annotations`** for text overlays positioned in data or paper coordinates.
- **`customdata` array** on traces stores per-point metadata for hover/click callbacks.
- **LaTeX-like math** via MathJax integration.
- Best for embedding interactive publication-quality charts as elements in the drawing canvas.
