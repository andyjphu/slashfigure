## What It Does
Python animation engine for explanatory math videos. Creates mathematical visualizations programmatically via scenes and Mobjects.

## Key for Our Project
Manim's scene/construct paradigm -- building a visual scene through sequential code operations -- maps directly to our "drawing actions generate metadata" model. Each `self.play()` call is analogous to a user drawing action. The code IS the structured description of the visual.
