# Kroki

- **Full Name:** Kroki -- Unified Diagram Rendering API
- **URL:** https://github.com/yuzutech/kroki / https://kroki.io/
- **Stars:** ~4,059
- **License:** MIT

## What It Does
Unified HTTP API rendering diagrams from text in 20+ formats (Mermaid, PlantUML, Graphviz, Svgbob, Ditaa, Pikchr, D2, etc.) to SVG/PNG/PDF.

## Key for Our Project
Demonstrates that a universal rendering API for text-based diagrams is feasible. Store diagram metadata in any text format, use Kroki-like rendering to display. Decouples metadata format from renderer. Could be used as a backend service.
