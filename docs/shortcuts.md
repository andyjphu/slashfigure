# Keyboard Shortcuts

## Tools
| Shortcut | Action |
|----------|--------|
| V | Select tool |
| R | Rectangle tool |
| T | Text tool |
| A | Arrow tool |
| P | Freehand (pen) tool |

## Selection
| Shortcut | Action |
|----------|--------|
| Ctrl+A | Select all visible layers in viewport |
| Ctrl+Shift+A | Select all layers (including hidden and off-screen) |
| Escape | Deselect all, switch to select tool |
| Click | Select element |
| Shift+Click | Toggle element in/out of selection |
| Click empty space | Deselect, start marquee |
| Drag empty space | Marquee select |

## Layers Panel
| Shortcut | Action |
|----------|--------|
| Click layer | Select that layer |
| Ctrl/Cmd+Click layer | Toggle layer in selection |
| Shift+Click layer | Range select (from last clicked to this) |
| Double-click layer | Rename layer |
| Drag layer | Reorder (blue insertion line shows position) |

## Edit
| Shortcut | Action |
|----------|--------|
| Ctrl+Z | Undo |
| Ctrl+Shift+Z | Redo |
| Ctrl+C | Copy selected (JSON + PNG to clipboard) |
| Ctrl+V | Paste with +20px offset |
| Ctrl+Shift+V | Paste in place (exact same position) |
| Delete / Backspace | Delete selected elements |

## File
| Shortcut | Action |
|----------|--------|
| Ctrl+S | Save (to existing file, or Save As if first time) |
| Ctrl+Shift+S | Save As (always shows file dialog) |
| Ctrl+O | Open .sf file |

## Viewport
| Shortcut | Action |
|----------|--------|
| Two-finger scroll | Pan |
| Pinch | Zoom |
| Space + drag | Pan |
| Middle mouse drag | Pan |

## Canvas Interaction
| Shortcut | Action |
|----------|--------|
| Double-click text | Edit text content (DOM overlay) |
| Double-click page tab | Rename page |
| Arrow keys (in input) | Increment/decrement value by step |

## Properties Panel
| Input | Action |
|-------|--------|
| Type number | Set absolute value |
| +N, -N, *N, /N | Relative math on current value |
| (expression) | Evaluate math: `50+25`, `(3*4)+2` |
| ~ | Shown for mixed values (multi-select) |
| ~+N, ~-N | Relative adjustment on mixed values |
| Drag opacity % | Scrub fill/stroke opacity |
