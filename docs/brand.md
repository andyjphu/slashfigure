# Brand

## Name
- **Product/company name:** \figure
- **When backslash is not possible:** slashfigure
- **Domain:** slashfigure.com

## Usage
- Prefer `\figure` in all communications where backslash rendering is supported
- Use `slashfigure` in URLs, file names, package names, GitHub org, and plain-text contexts where backslash may be escaped or invisible
- Domain: `slashfigure.com`
- NPM scope: `@slashfigure/` (if needed)
- GitHub: `slashfigure/` org or repo name
- PyPI helper package: `slashfigure-export` (future)

## File Naming
- `.sf` = single file project extension
- `slashfigure/` = default working directory name
- See `docs/features/project-format.md` for full format spec
